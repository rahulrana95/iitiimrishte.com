# Research Angle: Feature Inventory + Verification/Privacy

_Raw agent output. Target 403-blocked; from search snippets. Careful not to attribute Shaadi.com features to iitiimshaadi._

## Key findings — Matching / Search / Communication
- Niche portal for premier-institution alumni across disciplines; "Alma Mater Matters." [loving.is](https://www.loving.is/en/review/iitiimshaadi-com/)
- Freemium: free register + browse 10 profiles; paywall on all communication. Shortlist feature exists. [Google Play](https://play.google.com/store/apps/details?id=com.tycho.iitiimshadi&hl=en_US)
- Paid bundle: view full details, contacts (phone+email), messages + friend requests (interest system), unlock search filters, ID/keyword quick-search, profile-boost. [Google Play](https://play.google.com/store/apps/details?id=com.tycho.iitiimshadi&hl=en_IN)
- **Matching is filter/preference-driven, NOT described as algorithmic.** "Filters and preferences make it easy to find compatible profiles." [loving.is](https://www.loving.is/en/review/iitiimshaadi-com/)
- Contact reveal metered by plan (Lifetime = 500 contacts). Tiers: Testing ~$0.99, 3mo ~$249, 6mo ~$299, yearly ~$349, Lifetime ~$499/₹33,900+18% GST. [fee](https://iitiimshaadi.com/fee)
- Assisted/concierge via direct founder contact; "pay a sum to guarantee a good match." [ScoopWhoop](https://www.scoopwhoop.com/news/wedding-bells/)
- Native iOS + Android apps.

## Key findings — Verification / Privacy
- Education-first mandatory verification: docs submitted before database access. [loving.is](https://www.loving.is/en/review/iitiimshaadi-com/)
- Screens degree, ID card, marksheet — manual review, NOT institutional-email/registrar-API check. [Quora legit](https://www.quora.com/Is-IITIIMShaadi-com-legit)
- Manual photo approval to reduce fake accounts. [Google Play](https://play.google.com/store/apps/details?id=com.tycho.iitiimshadi&hl=en_US)
- Baseline visibility closed to public (only registered users see profiles). [loving.is](https://www.loving.is/en/review/iitiimshaadi-com/)
- Verified-active gap: ~30k of ~150k active/authenticated (~20%). [rocketreach](https://rocketreach.co/iitiimshaadicom-profile_b40e6989ffe62940)
- Privacy policy page exists (iitiimshaadi.com/users/privacy) but terms unretrievable.

## Gaps / Pain points
- Small active & verified pool, low interaction.
- Poor paid-conversion value ("null and void").
- App "extremely slow."
- Credibility/elitism reputational risk.
- **Unconfirmed/likely-absent features (evidence gap):** incognito/hidden browsing, "who viewed me," photo-blur/photo-password privacy, granular per-member blocking, horoscope/Kundli (Guna Milan) matching, algorithmic compatibility matching. (These exist on Shaadi.com — not to be attributed here.)
- Verification is manual document-based, **spoofable by forged documents** — not source-authenticated (no registrar/email/KYC).
- Data retention & third-party sharing terms not publicly retrievable.
