# Research Angle: Pricing, Tiers & Paywall Mechanics

_Raw agent output. Primary pages 403-blocked; from search snippets. Figures reported-not-verified; USD figures conflict across sources._

## Key findings
- Hard pay-to-contact wall: free = register + browse ~10 profiles; any communication requires paid membership. [fee page](https://iitiimshaadi.com/fee)
- **Web (INR):** Yearly ₹29,500; Six-Month ₹26,000 (unusually small gap → weak annual incentive). Plus service tax; "subject to change." [fee](https://iitiimshaadi.com/fee) · [loving.is](https://www.loving.is/en/review/iitiimshaadi-com/)
- **App-store (USD, inconsistent):** 3mo ~$249, 6mo ~$299, Yearly ~$399.99, Lifetime ~$499; another source "from $199." Treat as approximate. [App Store](https://apps.apple.com/in/app/iitiimshaadi-matchmaking-app/id1626456060)
- **Tiers gated by contact-unlock volume, not features:** unlock contacts of 20 / 50 / 150 / 500 profiles + unlimited friend requests/messages. [fee](https://iitiimshaadi.com/fee)
- Paid unlocks: full profile details, contacts (phone+email), messaging + friend requests, extra search filters, ID/keyword quick-search, profile-boost visibility. [fee](https://iitiimshaadi.com/fee)
- "Unlimited" messaging has anti-abuse caveat (suspension if misuse suspected).
- **Concierge/assisted service** priced opaquely (contact founder taksh@iitiimshaadi.com / +91-9811199800). Quote-on-request. [fee](https://iitiimshaadi.com/fee)
- Web renewal stated manual (not auto); app-store subs structurally auto-renewable (unconfirmed).
- "Refund & Cancellation" page exists but terms unretrievable; no iitiimshaadi-specific refund complaint found (results were Shaadi.com).
- One report: fee ~₹32,922 "until marriage." [ptcnews (from fraud agent)]
- Database ~150k total, ~30k active/authenticated, ~500 new/week — small pool for ₹26k–29.5k. [loving.is](https://www.loving.is/en/review/iitiimshaadi-com/)

## Gaps / Pain points
- Exact tier-to-price mapping unconfirmed; INR/USD figures conflict.
- Refund policy text not verified.
- Auto-renewal ambiguity on app.
- Assisted-service pricing fully opaque.
- Value-for-money #1 criticism (high price vs small reachable pool).
